Type Conversion
  • Implicit cast (widening): int i = 10; double d = i;
  • Explicit cast (narrowing): double d = 9.8; int i = (int)d;
  • Convert class: int i = Convert.ToInt32("123");
  • Parse method: int i = int.Parse("123");
  • TryParse method: bool ok = int.TryParse("123", out int i);
  • ToString method: string s = i.ToString();
// Implicit cast
    int i = 10;
    double d = i; // 10.0

    // Explicit cast
    double d2 = 9.8;
    int i2 = (int)d2; // 9

    // Convert class
    int n = Convert.ToInt32("123");

    // Parse/TryParse
    int p = int.Parse("456");
    bool ok = int.TryParse("789", out int val);

    // ToString
    string s = n.ToString();

Tip: Use TryParse for safe conversions from strings to numbers.

Loops
int[] numbers = {1, 2, 3, 4, 5};

// For loop
for(int i = 0; i < numbers.Length; i++) {
  Console.WriteLine(numbers[i]);
}

// Foreach loop
foreach(int num in numbers) {
  Console.WriteLine(num);
}

// While loop
int count = 0;
while(count < 5) {
  Console.WriteLine(count);
  count++;
}

// Do-while loop
int x = 0;
do {
  Console.WriteLine(x);
  x++;
} while(x < 3);
Switch Statement
int day = 3;
string dayName;

switch(day) {
  case 1:
    dayName = "Monday";
    break;
  case 2:
    dayName = "Tuesday";
    break;
  case 3:
    dayName = "Wednesday";
    break;
  default:
    dayName = "Unknown";
    break;
}

// Switch expression (C# 8.0+)
string result = day switch {
  1 => "Monday",
  2 => "Tuesday",
  3 => "Wednesday",
  _ => "Unknown"
};

C# Strings

String Concatenation
string first = "John";
string last = "Doe";

string name = first + " " + last;
Console.WriteLine(name); // => John Doe
String Interpolation
string first = "John";
string last = "Doe";

string name = $"{first} {last}";
Console.WriteLine(name); // => John Doe

Character Escape Sequences
EscapeMeaningNotes / Example
\" Double quote var s = \"She said \"Hello\"\";
\\ Backslash var path = \"C:\\\\Program Files\\\\App\";
\n Newline (LF) \"Line1\\nLine2\"
\r Carriage return (CR) Often used with \n as \r\n on Windows
\t Horizontal tab \"A\\tB\" → A B
\' Single quote Common in char literals: '\''
\0 Null character Rarely used in managed strings
\a Alert (bell) May have no audible effect on modern systems
\b Backspace Moves cursor back one position
\f Form feed Legacy printer control
\v Vertical tab Rarely used
\uXXXX Unicode (16-bit hex) \"\u03A9\" → Ω
\UXXXXXXXX Unicode (32-bit hex) For code points beyond BMP
\xH[H][H][H] Hex escape (variable length) Stops at non-hex unless 4 digits supplied

Tip: In verbatim strings @"...", backslashes are literal and newlines are preserved. To include a double quote, use two of them: @"He said ""Hi""".
